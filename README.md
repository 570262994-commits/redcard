# RedCard 🃏

小红书干货卡片生成器 | 3:4 精美内容卡片工具

---

## 📱 产品介绍

RedCard 是一款简洁高效的在线卡片生成工具，帮助内容创作者快速制作精美的 3:4 比例卡片，适用于小红书、微博、微信公众号等社交平台分享。

**核心价值**：用简单的 Markdown 语法，生成专业美观的知识卡片。

## 📖 使用指南

### 1. 输入内容

在左侧编辑器输入文本，支持以下 Markdown 语法：

| 语法 | 示例 | 效果 |
|------|------|------|
| 标题 | `# 标题文字` | 大标题 |
| 副标题 | `## 副标题文字` | 小标题 |
| 列表项 | `- 列表内容` | 带勾选图标 |
| 金句 | `> 引用的金句` | 引用框样式 |
| 作者 | `@作者名称` | 底部作者信息 |
| 加粗 | `**加粗文字**` | 粗体显示 |

### 2. 选择主题

点击顶部主题按钮切换卡片风格，鼠标悬停可实时预览效果。

### 3. 导出图片

- 点击「导出图片」按钮，选择清晰度
- 或直接在卡片上右键选择导出

## 🛠 技术栈

- **前端框架**：React 18 + Vite
- **样式方案**：Tailwind CSS v4
- **图标库**：Lucide React
- **导出引擎**：modern-screenshot
- **状态管理**：React Hooks + localStorage
