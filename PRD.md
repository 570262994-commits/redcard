# RedCard - 小红书干货卡片生成器

## 1. 产品概述

**产品名称**：RedCard  
**产品类型**：Web 在线工具  
**核心功能**：通过简单的文本输入，生成精美的 3:4 比例卡片图片，适用于小红书等社交平台分享  
**目标用户**：内容创作者、产品经理、自媒体从业者、知识博主

---

## 2. 功能清单

### 2.1 核心功能

| 功能 | 描述 | 状态 |
|------|------|------|
| 文本编辑器 | 支持 Markdown 语法输入（标题、副标题、列表、金句、作者） | ✅ |
| 实时预览 | 左侧编辑，右侧实时预览卡片效果 | ✅ |
| 5套主题 | 极客代码、蔚蓝灵感、蜜桃日记、梦幻视窗、纸墨极简 | ✅ |
| 主题悬停预览 | 鼠标悬停主题按钮时，卡片实时预览该主题效果 | ✅ |
| 一键导出 | 支持 2x/3x 清晰度导出 PNG 图片 | ✅ |
| 右键导出 | 支持右键卡片直接导出 | ✅ |
| 内容持久化 | localStorage 自动保存内容和主题选择 | ✅ |

### 2.2 编辑器功能

| 功能 | 描述 |
|------|------|
| 行号显示 | 编辑器左侧显示行号 |
| 工具栏 | 加粗、列表、金句、作者、表情快捷按钮 |
| 加载示例 | 一键加载示例模板（有确认弹窗保护） |
| 语法提示 | 底部显示 Markdown 语法提示 |

### 2.3 主题系统

| 主题 | 风格 | 适用场景 |
|------|------|----------|
| 极客代码 | 深色科技风 | 开发/技术类内容 |
| 蔚蓝灵感 | 浅蓝商务风 | 知识/商务类内容 |
| 蜜桃日记 | 粉色生活风 | 生活/女性向内容 |
| 梦幻视窗 | 玻璃拟态风 | 艺术/创意类内容 |
| 纸墨极简 | 米黄纸质风 | 文艺/阅读类内容 |

## 3. 技术架构

### 3.1 技术栈

- **前端框架**：React 18 + Vite
- **样式方案**：Tailwind CSS v4
- **图标库**：Lucide React
- **导出库**：modern-screenshot
- **持久化**：localStorage

### 3.2 项目结构

```
redcard/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    ├── App.jsx        # 主应用组件
    └── index.css      # Tailwind 入口
```

## 4. 交互设计

### 4.1 布局结构

```
┌─────────────────────────────────────────────────────┐
│  Header: Logo | 主题选择(5个) | 导出按钮           │
├─────────────────────┬───────────────────────────────┤
│                     │                               │
│   编辑器面板        │      卡片预览区域             │
│   (40% 宽度)        │      (60% 宽度)               │
│                     │                               │
│  - 工具栏           │   ┌─────────────────────┐   │
│  - 行号 + 文本框    │   │                     │   │
│                     │   │    3:4 卡片预览     │   │
│                     │   │                     │   │
│                     │   └─────────────────────┘   │
│                     │                               │
└─────────────────────┴───────────────────────────────┘
```

### 4.2 卡片尺寸

- **比例**：3:4 (宽度:高度)
- **预览尺寸**：360px × 480px
- **导出尺寸**：
  - 2x 清晰度：720 × 960 px
  - 3x 清晰度：1080 × 1440 px

## 5. Markdown 语法支持

| 语法 | 示例 | 效果 |
|------|------|------|
| 标题 | `# 标题` | 大标题 |
| 副标题 | `## 副标题` | 小标题 |
| 列表项 | `- 内容` | 带勾选图标列表 |
| 金句 | `> 金句内容` | 引用框样式 |
| 作者 | `@作者名` | 底部作者信息 |
| 加粗 | `**加粗文字**` | 粗体显示 |

## 7. 使用说明

1. **输入内容**：在左侧编辑器输入文本，支持 Markdown 语法
2. **选择主题**：点击顶部主题按钮切换，卡片实时预览
3. **预览效果**：鼠标悬停主题按钮可预览效果
4. **导出图片**：点击「导出图片」或右键卡片，选择清晰度后自动下载

---

## 8. 注意事项

- 导出图片时建议使用 3x 超清模式，可获得最佳清晰度
- 内容会自动保存，刷新页面不会丢失
- 加载示例会覆盖当前内容，请确认后再操作
